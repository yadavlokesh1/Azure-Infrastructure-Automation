# Azure Infrastructure Automation

Terraform project to deploy Windows Server infrastructure on Microsoft Azure.

## Resources Deployed
- Resource Group
- Virtual Network (VNet)
- Subnet
- Network Security Group (NSG) with RDP rule
- Public IP (Standard SKU)
- Network Interface
- Windows Server 2022 VM

## Technologies Used
- Terraform (IaC)
- Microsoft Azure
- Windows Server 2022

## How to Use
1. Install Terraform and Azure CLI
2. Run `az login`
3. Run `terraform init`
4. Run `terraform plan`
5. Run `terraform apply`
